WEB 3.0


Web 3.0 adalah internet generasi ketiga yang akan datang di mana situs web dan aplikasi akan dapat memproses informasi dengan cara yang cerdas melalui teknologi seperti machine learning, big data, teknologi buku besar terdesentralisasi (DLT), dll.


Web 3.0 awalnya disebut Web Semantik oleh penemu World Wide Web Tim Berners-Lee, dan ditujukan untuk menjadi internet yang lebih mandiri, cerdas, dan terbuka.


Selain itu, definisi dari Web 3.0 dapat diperluas menjadi data akan saling berhubungan dengan cara yang terdesentralisasi di mana data sebagian besar disimpan dalam repositori terpusat.


Selanjutnya, pengguna dan mesin akan dapat berinteraksi dengan data. Namun agar hal ini terjadi, program perlu memahami informasi baik secara konseptual maupun kontekstual.


Maka begitu, dua landasan utama Web 3.0 adalah web semantik dan kecerdasan buatan (AI).
